# LLM-Based-Trip-Planning-Implementation

AI Travel Planner is a comprehensive travel planning application that leverages artificial intelligence to create detailed, personalized travel itineraries. The system integrates multiple data sources including real-time weather information, local attractions, restaurants, transportation options, and currency conversion to generate complete travel plans with expense breakdowns and daily budgets.

Built with FastAPI backend and Streamlit frontend, the application utilizes LangGraph for agentic workflows and supports multiple LLM providers including Groq and OpenAI. It features a modular tool system for weather forecasting, place search, expense calculation, and currency conversion, providing users with thorough travel planning capabilities in a single, cohesive interface.
